# El Romeral - Project Context

## Overview
- **Project**: El Romeral - Wedding & Events Venue in Guadalajara
- **Tech Stack**: Next.js 16 (React 19), TypeScript, Tailwind CSS 4, Supabase, Resend
- **Hosting**: Vercel

## Authentication & Security

### Middleware (`middleware.ts`)
- Updates Supabase session on every request
- Protects `/admin/*` routes (redirect to login if no session)
- Protects `/api/admin/*` routes (401 if no auth)
- Exceptions: `/admin/login`, `/admin/setup`, `/api/admin/create-user`

### Auth Cookie Detection
- Looks for cookies named `sb-*-auth-token`
- Session managed by Supabase SSR proxy

### Environment Variables Required
- `NEXT_PUBLIC_SUPABASE_URL` - Supabase URL
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` - Anonymous key
- `SUPABASE_SERVICE_ROLE_KEY` - Service role (server-side only)
- `RESEND_API_KEY` - Email API key
- `ADMIN_SETUP_KEY` - Secret key for admin setup
- `NEXT_PUBLIC_SITE_URL` - Site URL (default: `https://elromeral.com.mx`)

## Important Notes

1. **RLS Policy**: Supabase Row Level Security is used (see `fix-blocked-dates-rls.sql`)
2. **Email**: All emails sent via Resend, configured in API routes
3. **PDF**: Multiple PDF generation endpoints - consolidate if redundant
4. **Quote Progress**: Tracked via `quote-progress.ts` utility
5. **Admin Setup**: Separate from normal login, uses `ADMIN_SETUP_KEY`
6. **Multiple Quote Flows**: Regular, adicionales (additional), personalizada (custom), diurnos (daytime)
7. **Theme Support**: Via `next-themes` - light/dark mode support

## Common Tasks

### Adding a New Admin Page
1. Create directory under `app/admin/[feature]/`
2. Add `page.tsx` and/or `layout.tsx`
3. Add corresponding component in `components/admin/[feature]/`
4. Middleware automatically protects with auth
5. Link in admin navigation

### Adding a New API Endpoint
1. Create file under `app/api/[feature]/route.ts`
2. Export POST/GET/etc. handlers
3. If admin API: middleware requires auth
4. Use Supabase client from `@/lib/supabase/`

### Quote/Email Flow
1. Generate quote (configurador)
2. Send email via `send-cotizacion-email/` or `send-quote-email/`
3. Export to PDF via `export-pdf/` or similar
4. Track status in database

## Notes for Future Development
